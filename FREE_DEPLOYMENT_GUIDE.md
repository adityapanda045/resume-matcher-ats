# 🚀 Babu's Zero-Cost Deployment Guide (100% Free)

Babu, follow these steps to get your project live on the internet without spending any money.

## 1. Get your Free API Key
- Go to [Google AI Studio](https://aistudio.google.com/).
- Create a **Free API Key** for Gemini.
- *Cost: $0*

## 2. Push to GitHub
- Create a new public repository named `resume-matcher-ats`.
- Upload `app.py`, `requirements.txt`, `README.md`, and `PROJECT_STORY.md`.
- *Cost: $0*

## 3. Deploy to Streamlit Cloud
- Go to [streamlit.io/cloud](https://streamlit.io/cloud).
- Connect your GitHub account.
- Click **"New App"** and select your repository.
- **IMPORTANT (Secrets)**: 
  - Before clicking Deploy, go to **Advanced Settings**.
  - In the **Secrets** box, paste this:
    ```toml
    GOOGLE_API_KEY = "PASTE_YOUR_GOOGLE_API_KEY_HERE"
    ```
- Click **Deploy**.
- *Cost: $0*

## 4. Share with Recruiters
- Once the app is live, you will get a URL (like `babu-ats-data-model.streamlit.app`).
- Add this link to your **LinkedIn Profile** and **Resume**.
- Tell recruiters: *"I built this NLP Data Dashboard to optimize the recruitment process using Generative AI."*

## 5. Timeline for Success
- **Day 1**: Deploy.
- **Day 2-5**: Update LinkedIn and Resume with the link.
- **Day 6-30**: Apply for Data Analytics roles and demo this project during interviews.

**You can do this! This project proves you are ready for a professional Data role.**
